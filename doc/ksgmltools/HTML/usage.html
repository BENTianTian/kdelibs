2.2. Running the Tools

To convert files from DocBook format to HTML format, use the command :

db2html -d /usr/lib/sgml/stylesheets/kde.dsl mydocs.docbook

Jade will chug away, and if your document has no errors you will get a set of HMTL files named after the Chapters in your document. To get a good feel for how this tool works, try using the db2html tool on the source file for this document (crash-course.docbook) and on the documentation template (template.docbook).

If you do get errors, read through the error log and correct errors from the beginning of the list first. Often, an early error such as an unclosed tag will cause a lot more errors to occur later in the document.

Other tools may let you convert docbook files to other formats:

  • db2ps => PostScript

  • db2pdf => PDF

  • db2rtf => Rich Text Format

  • db2dvi => TeX DVI
